# 🔍 Forensic Audit Automation Tool

## Overview
This project demonstrates an automated approach to financial auditing using Python and the Pandas library. It is designed to handle messy, multi-currency transaction data.

## Key Features
* **Currency Normalization:** Automatically converts KES to USD based on real-time or fixed rates.
* **Duplicate Detection:** Identifies potential double-payments or fraud patterns via Transaction ID analysis.
* **Compliance Flagging:** Highlights "Ghost Vendors" (missing data) and high-risk status items.

## Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy
* **Environment:** Google Colab / VS Code
